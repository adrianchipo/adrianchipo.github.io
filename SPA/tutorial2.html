Estilismo

Para preparar el escenario, agregaremos un degradado sutil a la pantalla para que parezca una cabeza blanca curva. Para esto usaremos un degradado radial en el body

                        
                            body {
                                background: radial-gradient(circle at center, #fff, #fff 50%, #aaa);
                                background-size: 100% 100%;
                                background-repeat: no-repeat;
                                height: 100vh;
                              }

A continuación colocamos el rostro en el centro de la página. La boca es una línea negra simple, y la crearemos usando un borde.

                        
                            .baymax {
                                border-bottom: 1.5em solid #000;
                                position: absolute;
                                top: 50%;
                                left: 50%;
                                width: 50%;
                                transform: translate(-50%, -40%);
                              }
                        
                    
                        
                            ul, li {
                                padding: 0;
                                margin: 0;
                                list-style: none inside;
                            }
                            ul.slider {
                                position:relative; /*Necesitamos que el ul sea relativo*/
                            }
                            ul.slider li {
                                opacity: 0; /*Ocultamos todos los <li>*/
                                transition: opacity .5s;
                                position: absolute; /*Posicionamos en absoluto los <li>*/
                                left: 0px;
                                top: 0px;
                                margin:50px;
                                padding:50px;
                                border:2px solid red;
                                background:#fff;
                            }
                            ul.slider li:first-child {
                                opacity: 1; /*Mostramos el primer <li>*/
                            }

La primera propiedad establece un borde negro de ancho 1.5em. Luego posiciona la línea usando posicionamiento absoluto, empujándola hacia abajo 50% y frente a la izquierda 50%. Estos valores del 50% se relacionan con el tamaño del body elemento contenedor() El problema que tenemos ahora es que el elemento comienza a la mitad de la pantalla. Está descentrado. Para compensar esto, usamos una transformación para tirar del elemento a la izquierda el 50% de su ancho y hacia arriba el 40% de su ancho.

Añadiendo los ojos

Haremos uso de las before y after los pseudo-elementos a añadir a los ojos de la cara. Esto no necesitará HTML adicional y es completamente manejado por CSS. 

                        
                            .baymax::before {
                                background: #000;
                                border-radius: 50%;
                                content: "";
                                position: absolute;
                                width: 12em;
                                height: 12em;
                                left: -9em;
                                top: -6em;
                                transform: skewX(-4deg);
                              }
                              
                              .baymax::after {
                                background: #000;
                                border-radius: 50%;
                                content: "";
                                position: absolute;
                                width: 12em;
                                height: 12em;
                                right: -9em;
                                top: -6em;
                                transform: skewX(4deg);
                              }

Cada uno de estos pseudoelementos tiene un fondo negro y un radio de borde del 50% para hacerlos circulares. Cada uno se coloca al final de la boca y, por último, tiene una skew transformación para que parezca que están un poco inclinados hacia atrás. El resultado debería verse así:

Batería baja

Hay una escena divertida en la película donde la batería de Baymax se agota. Se tambalea y sus párpados se caen. Podemos usar una combinación de degradados de fondo y animación para crear el efecto. Primero queremos darle al fondo dos colores. Negro para la parte del ojo que está abierta y blanco para el párpado. La parte blanca debe colocarse fuera de los ojos al principio, luego la animaremos para que los párpados se caigan. 

                        
                            .baymax::before {
                                background: linear-gradient(to bottom, #efefef, #efefef 50%, #000 50%, #000);
                                background-position: 0 -100%;
                                background-size: 200% 200%;
                                ...
                              }
                              
                              .baymax::after {
                                background: linear-gradient(to bottom, #efefef, #efefef 50%, #000 50%, #000);
                                background-position: 0 -100%;
                                background-size: 200% 200%;
                                ...
                              }

Estamos agregando un degradado lineal de fondo, lo hicimos dos veces la altura del contenedor y luego lo colocamos de manera que la mitad superior quede fuera del contenedor. Con los dos degradados de fondo en su lugar, podemos agregar animación keyframespara controlar el movimiento de los párpados. 

                        
                            @keyframes blink {
                                0%, 50% {
                                  background-position: 0 100%;
                                }
                                85%, 95% {
                                  background-position: 0 75%;
                                }
                                100% {
                                  background-position: 0 100%;
                                }
                              }

La animación keyframeses una forma de describir una serie de fotogramas, utilizando un porcentaje. El porcentaje se relaciona con cuánto tiempo ha durado la animación. Entonces, un porcentaje del 50% significa que se encuentra en la mitad de la animación. De esta manera, podemos configurar el fondo para que permanezca activo hasta la mitad, luego entre el 50% y el 85%, se mueve hacia abajo y luego vuelve al final de la animación. El siguiente paso es decirle a los pseudoelementos que utilicen estos fotogramas clave de animación. Agregue la animationpropiedad a los estilos existentes. 

                        
                            .baymax::before {
                                animation: blink 6s infinite;
                                ...
                              }
                              
                              .baymax::after {
                                animation: blink 6s 0.1s infinite;
                                ...
                              }

Aquí le hemos dicho al navegador que use la blinkanimación en cada elemento. La duración de la animación se establece en seis segundos y se repetirá infinitamente. Una propiedad adicional está en el segundo ejemplo. El 0.1sdespués 6s le dice al navegador que retrase esa animación en 0,1 segundos. Esto crea el efecto de que el segundo ojo se cierra un poco más tarde que el primero. Se suma al efecto cansado y caído y lo hace parecer un poco más humano
