El efecto Ken Burns usando animaciones CSS

¿Que es?

Es una panorámica y zoom lentos de una imagen estática . Si bien esa descripción suena muy aburrida, puede crear muchos efectos geniales con ella, como probablemente haya visto en innumerables documentales, programas de crímenes y cualquier otra cosa en la que necesite resaltar algo en una imagen estática durante un tiempo.

CSS y transformaciones

En CSS, las estrellas de este efecto son las transformaciones translate3d y scale3d que se encargan de hacer una panorámica y hacer zoom en la imagen. La transformación translate3d mueve la imagen hacia la izquierda, derecha, arriba o abajo

Punto de partida

Cuando combina ambas transformaciones junto con una animación lenta, tiene el efecto Ken Burns. Comprender esto es la parte difícil. La parte fácil es escribir el CSS

Si desea seguir activamente, cree una nueva página HTML y agregue las siguientes líneas de cosas en ella: 

                        
                            
                                
                            <!DOCTYPE html>
                            <html>
                             
                            <head>
                                <title>The Ken Burns Effect in CSS</title>
                                <style>
                                    h1 {
                                        font-family: monospace;
                                        font-size: 2.1em;
                                        color: #3399FF;
                                    }
                                    body {
                                        padding: 10px;
                                        background-color: #F4F4F4;
                                    }
                                    #imageContainer {
                                        background-color: #333;
                                        width: 450px;
                                        height: 300px;
                                        overflow: hidden;
                                        border: 2px #333 solid;
                                    }
                                </style>
                            </head>
                             
                            <body>
                             
                                <h1>The Ken Burns Effect in CSS</h1>
                             
                                <div id="imageContainer">
                                    <img src="//www.kirupa.com/images/example_sm2.jpg">
                                </div>
                             
                                <script src="//www.kirupa.com/prefixfree.min.js"></script>
                            </body>
                             
                            </html>

Si obtiene una vista previa de esta página en su navegador, verá una página simple con una imagen estacionaria. El único detalle interesante es lo que ves con la propia imagen. Nuestra imagen está recortada (también conocida como enmascarada) por los límites de su contenedor principal gracias a la siguiente línea de CSS:

                        
                            #imageContainer {
                                background-color: #333;
                                width: 450px;
                                height: 300px;
                                overflow: hidden;
                                border: 2px #333 solid;
                            }

Esto significa que podemos hacer zoom y desplazar la imagen libremente sin ninguna preocupación en el mundo. La ventana gráfica es mantenida por el elemento div principal que contiene la imagen.

Agregar la animación CSS

Nuestra animación CSS es bastante sencilla. Utilizará diferentes valores de translate3d y scale3d para cambiar nuestra imagen. Comencemos por traducir eso en valores de fotogramas clave, así que continúe y agregue nuestros fotogramas clave (llamados kenburns ) hacia la parte inferior de su etiqueta de estilo : 

                        
                            @keyframes kenburns {
                             0% {
                               opacity: 0;
                            }
                             5% {
                             opacity: 1;
                            }
                              95% {
                              transform: scale3d(1.5, 1.5, 1.5) translate3d(-190px, -120px, 0px);
                             animation-timing-function: ease-in;
                             opacity: 1;
                             }
                            100% {
                                transform: scale3d(2, 2, 2) translate3d(-170px, -100px, 0px);
                               opacity: 0;
                            }
                            }

La acción real ocurre comienza en la marca del 95%. Preste atención a cómo se ve nuestra declaración de transformación : 

                        
                            transform: scale3d(1.5, 1.5, 1.5) translate3d(-190px, -120px, 0px);

Hacemos zoom en la imagen en un 50% usando scale3d , y cambiamos la imagen 190 píxeles horizontalmente y 120 píxeles verticalmente usando translate3d . Continuamos más de esto en la marca del 100% también, donde hacemos zoom y desplazamos un poco más para ajustar el efecto. Lo último que queda es definir nuestra animación CSS que vincule nuestros fotogramas clave kenburns con un valor de duración. Agregue las siguientes líneas dentro de su etiqueta de estilo justo encima de la declaración de fotogramas clave: 

                        
                            #imageContainer img {
                                animation: kenburns 20s infinite;
                              }

EEspecificamos que nuestra animación se ejecutará durante 20 segundos y repetirá los fotogramas clave de kenburns para siempre. Si guarda estos cambios y obtiene una vista previa de la página en su navegador, verá el efecto Ken Burns en su pantalla donde la imagen se desplaza lentamente hacia la oveja de aspecto culpable que todas las otras ovejas están mirando.
